====== PLOSTRUCTURE ======
 Remote Control (Live Table Actions)                                                               
                                                                     
	- app.py - Flask backend API that receives table snapshots from n4p.js, manages command queue,
	merges multi-seat data
	- index.html - Remote Control UI with 9-seat table display, action buttons
	(FOLD/CHECK/CALL/RAISE/CASHOUT)
	- n4p.js - Browser-injected scraper that reads poker site DOM, sends snapshots to backend every
	2s, polls for commands every 250ms

	Equity Engine (Math/Calculations)

	- engine.html - Hand analysis UI for equity calculations
	- plo4_6max.py, plo5_6max.py, plo6_6max.py, plo7_6max.py - PLO variant calculators in
	/opt/plo-test/engine/scripts/
	- plo4_disparity_simulator.py - Disparity analysis tool
	- equity_routes.py - Flask routes for equity calculations

	Hand Collector (History Import)

	- server.py - Hand Collector backend API
	- hero_only_extractor.py - Extracts only Hero hands from history
	- hero_aggregator.py - Aggregates Hero statistics
	- collector_routes.py - Flask routes for collector endpoints

	Frontend Shell (Unified UI)

	- frontend/app.py - Main Flask app with multi-tab navigation
	- frontend/templates/remote.html, engine.html, collector.html, bots.html, tables.html, shell.html,
	 logs.html, ledger.html - Tab templates
	- frontend/static/js/emoji.js - Double-emoji visual system (🎮🃏, 🧮🔢, etc.)
	- frontend/static/js/shell.js - Sidebar navigation and layout
	- frontend/static/js/router.js - Client-side routing
	- frontend/static/js/remote.js, engine.js, collector.js, bots.js, tables.js - Page-specific logic

	Bot Automation

	- poker_bot_human.py - Selenium bot with human-like behavior (typing with typos, random pauses)
	- deploy-bot.sh - Single bot deployment
	- deploy-bots-multi-env.sh - Multi-bot deployment (8 bots, configurable credentials)
	- check-all-bots.sh - Check bot statuses
	- monitor-bot.sh - Monitor specific bot

	Deployment Scripts

	- deploy-n4p.sh - Deploy n4p.js + Remote Control to production
	- deploy-collector.sh - Deploy Hand Collector
	- deploy-seat-status.sh - Deploy Equity Engine
	- auto-deploy-cloud.sh - Full cloud setup (SSL, nginx, Flask)
	- snapshot-n4p.sh, restore-n4p.sh - Backup/restore Remote Control

	Documentation

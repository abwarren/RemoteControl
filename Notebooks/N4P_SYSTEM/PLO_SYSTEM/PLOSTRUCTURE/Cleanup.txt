  ---                                                                                                                                                                                                       
  Key Decisions Captured
                                                                                                                                                                                                            
  1. "Scraper" over "Agent" ✅
                                                                                                                                                                                                            
  - n4p.js reads DOM, builds snapshots, sends data                                                                                                                                                          
  - Not autonomous reasoning/decision-making                                                                                                                                                                
  - New name: Table Scraper or Browser Scraper                                                                                                                                                              
                                                                                                                                                                                                            
  2. "Remote Control" over "Control Panel" ✅                                                                                                                                                               ─
                                                                                                                                                                                                            
  - Matches actual function (live table control)                                                                                                                                                            
  - Distinguishes from admin/settings dashboards
  - Clear purpose: live action execution                                                                                                                                                                    
                                                                                                                                                                                                            
  3. Equity Engine ≠ Remote Control ✅                                                                                                                                                                      ─
                                                                                                                                                                                                            
  - Equity Engine = math, calculations, hand parsing, equity                                                                                                                                                
  - Remote Control = live seats, buttons, actions, UI
  - NEVER MERGE THESE BACKENDS (critical architecture decision)

  Architecture Guardrails                                                                                                                                                                                   
                                                                                                                                                                                                            
  ✅ Keep Separate Forever
                                                                                                                                                                                                            
  - Remote Control backend ≠ Equity Engine backend                                                                                                                                                          
  - Different purposes, data models, performance profiles                                                                                                                                                   
  - Remote Control = fast, real-time, in-memory                                                                                                                                                             
  - Equity Engine = slow, computational, batch-capable                                                                                                                                                      
                                                                                                                                                                                                            
  ❌ Never Merge                                                                                                                                                                                            ─
                                                                                                                                                                                                            
  - Remote Control + Equity Engine into one app.py                                                                                                                                                          
  - Table Scraper into Remote Control UI
  - Hand Collector into Remote Control                                                                                                                                                                      
                                                                                                                                                                                                            
  ✅ Can Share                                                                                                                                                                                              ─
                                                                                                                                                                                                            
  - Common utilities (auth, logging, card parsing)                                                                                                                                                          
  - Static assets (CSS, fonts, icons)
  - Database (if added later)
